## Installation

To get started with Shopping-App-UI, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/John-Wassef/Shopping-App-UI.git
    ```
2. Navigate to the project directory:
    ```sh
    cd Shopping-App-UI
    ```
3. Install the dependencies:
    ```sh
    flutter pub get
    ```
4. Run the app:
    ```sh
    flutter run
    ```
